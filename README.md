# moody
Moody is a box with 3 buttons to get people's moods every day.

# User Stories
- As a user, I can provide my mood by clicking on a (physical) button 
- As a user, I can see the stats of the weekdays

# Roadmap
- [ ] Detect a click on a button

- [ ] Setup database
